# dog-api

Conexión Api con https://dog.ceo/dog-api/documentation/

## Comenzando 🚀

Solo necesitas la instación global de `create-react-native-app`

### Instalación 🔧

`yarn install` y `npx pod-install` 


```
yarn run ios
```

_O_

```
yarn run android
```
